# Monitor de Tanques IoT - Electiva
Proyecto de gestión de nivel de agua usando ESP32-S3 y sensor Modbus RS485.

## Conexiones Hardware
- **Sensor RS485:** TX(7), RX(8), EN(21)
- **Módem CAT 1:** TX(17), RX(18), KEY(4)

## Stack
- Backend: Node.js + Express
- BD: MongoDB
- Protocolo: MQTT
